# Sol Shop
This is an example Monolith application with few basic concepts of an e-commerce system.

### Dev Setup
```sh
make gen-ent
go run cmd/main.go
```

# Stack
Backend written in `go` and mainly shaped around `echo` and `ent` frameworks.
